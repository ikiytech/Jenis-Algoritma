7 Categories of Algorithms

Explore different algorithm types and their examples

1. Algoritma Berdasarkan Tujuan

  • Search Algorithm: Linear Search, Binary Search
  • Sorting Algorithm: Bubble Sort, Quick Sort, Merge Sort
  • Shortest Path: Dijkstra's, A* Algorithm, Bellman-Ford

2. Algoritma Berdasarkan Pendekatan

  • Greedy Algorithm: Kruskal, Prim
  • Divide and Conquer: Merge Sort, Quick Sort
  • Dynamic Programming: Fibonacci, Knapsack
  • Backtracking: N-Queens, Maze Solving
  • Brute Force: Exhaustive Search

3. Algoritma Kriptografi

  • RSA Algorithm: Public Key Encryption
  • AES: Advanced Encryption Standard

4. Algoritma Pencarian Heuristik

  • Genetic Algorithm: Evolutionary Optimization
  • Simulated Annealing: Optimization Heuristic

5. Algoritma Graf

  • Breadth-First Search (BFS): Level Order Traversal
  • Depth-First Search (DFS): Depth Order Traversal

6. Algoritma Pembelajaran Mesin

  • K-Nearest Neighbors (KNN): Classification
  • Decision Tree: Decision Making
  • Neural Networks: Deep Learning
  • Support Vector Machine (SVM): Hyperplane Classification

7. Algoritma Optimisasi

  • Linear Programming: Constraint Optimization
  • Gradient Descent: Function Minimization
